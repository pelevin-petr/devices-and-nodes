# Editor of the list of devices and their accounting nodes

## Installation 🔧

### Clone the Repository:

```bash
git clone https://github.com/pelevin-petr/devices-and-nodes/tree/dev
cd ParkGorkStorageAPI
```

### Install Dependencies:

```bash
npm install
```

### Usage⚙️

```bash
npm run dev
```
